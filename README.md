# 나만의 브이로그 자동 제작 서비스

**개발 배경**

일상을 영상으로 기록·추억하고 싶은 MZ 세대가 증가하고있지만 영상 편집의 경우 오랜 시간이 소요되고 어려움

<br>

**특징**

1. AI 모델을 이용해 요약된 영상과 썸네일, 해시태그를 사용자에게 제공
2. 요약된 나만의 영상을 피드 페이지를 통해 다른 사람들과 공유하고 댓글로 소통 가능

<br>

**담당 역할**

썸네일 생성
